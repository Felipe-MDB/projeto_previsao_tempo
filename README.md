# 🌤 Weather App

Um aplicativo de clima desenvolvido com **HTML, CSS e JavaScript**, consumindo dados em tempo real da API Open-Meteo.

O projeto permite pesquisar o clima de qualquer cidade, visualizar informações meteorológicas atuais e consultar a previsão dos próximos dias através de uma interface moderna e responsiva.

---

## 🚀 Funcionalidades

* 🔍 Busca de clima por nome da cidade
* 📍 Consulta usando a localização atual do usuário
* 🌡 Exibição da temperatura atual
* 💧 Exibição da umidade relativa do ar
* 🌬 Exibição da velocidade do vento
* 📅 Previsão para os próximos 7 dias
* 🌙 Modo escuro (Dark Mode)
* 💾 Armazenamento da última cidade pesquisada
* 📱 Layout responsivo para dispositivos móveis
* ⚡ Atualização em tempo real via API

## 📸 Funcionalidades Principais

### Busca por Cidade

Digite o nome de uma cidade e visualize:

* Temperatura atual
* Umidade do ar
* Velocidade do vento
* Data e hora da atualização

### Geolocalização

Permite consultar automaticamente o clima da localização atual do usuário.

### Previsão para 7 Dias

Exibe temperaturas máximas e mínimas dos próximos dias.

### Tema Escuro

Alterna entre modo claro e escuro para melhor experiência visual.

## ⚙️ Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Felipe-MDB/projeto_previsao_tempo.git
```

### 2. Entre na pasta do projeto

```bash
cd projeto_previsao_tempo
```

### 3. Abra o projeto

Abra o arquivo `index.html` diretamente no navegador.

Ou utilize a extensão **Live Server** do VS Code.
